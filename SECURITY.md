# Security And Privacy

This project is intended to run locally.

## Sensitive Data

Do not commit:

- real resumes
- candidate personal details
- immigration documents
- generated application packets
- SQLite databases
- ingest logs with private notes
- API keys or cookies

Use local-only files for private data. The `.gitignore` already excludes common runtime folders and local profile overrides.

## Platform Safety

- Do not scrape LinkedIn or Indeed.
- Do not add unattended auto-submit.
- Keep browser autofill human-reviewed.

## Reporting Issues

For this personal repository, open a GitHub issue or fix directly in a branch.
