# Roadmap

## Phase 1-2: Current

- Job ingestion.
- Deduping.
- Deterministic scoring.
- Daily review queues.
- Pipeline summaries.
- Unit tests.

## Phase 3: Resume Tailoring

- Create `truth_bank.schema.json`.
- Build a private `truth_bank.json` from the candidate resume.
- Generate selected-job Claude packets.
- Produce tailored Markdown resumes with truth-bank claim IDs.
- Add non-fabrication checks.

## Phase 4: Application Workflow

- Track selected/applied/interview/rejected status.
- Add export/import bridge to Simplify or Teal if useful.
- Keep human final submit.

## Phase 5: Optimization

- Weekly analytics by source, role, sponsorship signal, and response rate.
- H-1B company history lookup.
- Better ATS source discovery.
- Optional DOCX/PDF resume generation.
